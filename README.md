# 小黑盒发布模板助手

一个 Tampermonkey(油猴)用户脚本,为[小黑盒](https://www.xiaoheihe.cn/)创作者发布页注入模板面板,支持保存常用发布配置并一键填充,免去每次发帖重复填写标题、正文、社区、话题、声明等信息的麻烦。

## 功能

- 📋 **模板管理**:保存、覆盖、删除多套发布模板,模板按发布类型(图文/文章/视频)分开存储,互不干扰;模板名称可单独配置和修改,留空时自动以标题命名
- 📥 **读取页面内容**:一键把当前编辑器里已填写的标题、正文、社区、话题、声明抓取到模板表单,编辑后可直接保存为模板
- 📤 **导出 JSON**:将当前表单配置导出为 JSON 文件,方便备份和分享
- ⚡ **一键填充**:自动填入标题、正文(换行自动分段)、关联社区(≤2 个)、关联话题(≤5 个)
- 🔍 **从列表选**:编辑模板时可点击「从列表选」按钮,直接打开小黑盒官方的社区/话题选择弹窗(含推荐列表、搜索、分类 tab),点选名称写入模板,不会真正添加到当前草稿,选完点弹窗右上角 × 结束
- 📝 **内容声明**:支持原创/转载声明的完整联动填充
  - 原创:转载权限(禁止转载 / 注明出处)
  - 转载:转载来源(站内/站外)、媒体名称、授权声明
  - 其他声明:AI 辅助创作、剧透、虚构演绎
- 💾 **本地存储**:模板通过 Tampermonkey 的 `GM_setValue` 保存在本地,并同时镜像到站点 `localStorage`——重新安装脚本导致 GM 存储被重置时会自动从镜像恢复,不上传任何数据

## 使用方法

1. 打开小黑盒创作中心的图文发布页(`https://www.xiaoheihe.cn/creator/editor/draft/image_text/...`)
2. 页面右下角会出现「📋 图文模板」悬浮按钮,点击展开面板
3. 填写模板内容:
   - **新建模板**:「选择已存模板」保持「— 新建 —」,填写字段后点击「💾 保存模板」
   - **使用模板**:在下拉框中选择已存模板,点击「⚡ 填入编辑器」自动填充发布页
   - **修改模板**:选中已有模板并编辑字段后保存,会覆盖原模板
4. 填充完成后检查内容,手动上传图片并发布

> 注意:图片/视频文件无法通过模板填充,需要手动上传。

## 工作原理

脚本采用**适配器模式**组织代码,每种发布类型对应一个适配器,声明式地定义面板字段(`fields`)和填充逻辑(`apply`):

- 通过 `document.execCommand` 向 ProseMirror 编辑器写入标题和正文
- 通过原生 `value` setter + `input` 事件填充表单输入框,触发前端框架响应
- 通过模拟点击完成下拉选择、社区/话题搜索弹窗的操作

如需支持文章/视频发布页,只需参照 `imageTextAdapter` 补全对应适配器的 `fields` 与 `apply` 即可,面板 UI 会自动复用。
